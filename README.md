# YeoJeong
국내 여행지 추첨기 : domestic destination drawing machine

Go(Golang) 백엔드와 Vanilla JS 프론트엔드로 구성된 **웹 기반 국내 여행지 랜덤 추첨 시스템**입니다. 
전국 시/군 단위 또는 울산 지역의 읍/면/동 단위를 선택하여 무작위로 여행지를 추천받을 수 있으며, 추첨된 목적지를 네이버 지도 및 유튜브 검색과 바로 연동할 수 있습니다.

---

## ✨ 주요 기능
- **두 가지 추첨 옵션**: '전국 시/군' 범위 또는 '울산 읍/면/동' 범위 선택 가능
- **시각적 효과**: 추첨 버튼 클릭 시 긴장감을 주는 플레이스 홀더 롤링 애니메이션 제공
- **외부 채널 연동**: 추첨 완료 후 해당 목적지의 **네이버 지도(위치 확인)** 및 **유튜브(Vlog/여행 정보)** 검색 링크 자동 생성
- **경량 백엔드**: 별도의 프레임워크 없이 Go 표준 라이브러리(`net/http`)만을 사용한 초경량 REST API 서버
